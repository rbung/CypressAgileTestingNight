Slides are available here : <https://slides.com/rodolphebung/cypress-agile-testing-night>

## Pre-requisites

- A NodeJS installation

## Installation

```bash
npm install
```

## Run the web application

```bash
npm start
```

## Open Cypress UI

```bash
npm run cypress:open
```

## Run Cypress tests in headless mode

```bash
npm run cypress:run
```
